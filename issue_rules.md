## Глоссарий
### Программный модуль
Приложение, которое содержит в себе:
  + Документы
  + Область навигации
  + Представления
  + Регистрационные карточки
  + Отчеты

### Документ
Запись в ПМ, в котором хранятся данные.

### Область навигации
Набор ссылок на предсталения текущего ПМ.  
Также может содержать в себе ссылки на другие ПМ

### Представление
Табличное отображение документов приложения, в котором каждая строка - это документ, а каждый столбец - это значение поля документа.

### Регистрационная карточка
Форма ввода данных в документ и ее отображения.
Бывает в 2 состояниях:
  + Чтения
  + Редактирования
#### Панель действий

#### Атрибут
Название и поле для ввода данных.  

##### Текстовый
Может содержать в себе любые печатные символы.

##### Числовой
Может содержать в себе только цифры.

##### Дата и время
Может содержать в себе дату и/или время.

##### Ссылочный
Содержит в себе ссылку на другой документ.

## Ошибка
Тип заявки, с помощью которой создается запрос на **исправление** поведения Системы, которое противоречит ранее утвержденной [Истории](#история)

### Запрос на исправление ошибки в РК
1. В ПМ "<Название ПМ>", в РК "<Название РК>", наблюдается ошибочное поведение Системы:
  + Пользователь открыл РК "<Название РК>" в режиме (чтения|редактирования)
  + Пользователь нажал на <Полный/путь/к/элменту/дизайна/Название элемента>
  + Система выполнила операцию ...
2. Ссылка на Историю добавления или изменение функциональности, в которой наблюдается ошибка
3. `[Файл лога]`

### Запрос на исправление ошибки в Представлении
1. В ПМ "<Название ПМ>", в представлении "<Название представления>", наблюдается ошибочное поведение Системы:
  + Ошибка в действии:
    + Пользователь открыл представление <Название представления>
    + Пользователь нажал на <Полный/путь/к/действию/Название действия>
    + Система выполнила операцию ...
  + Ошибка в отображении:
    + Система неправильно отобрала документы
    + Столбцы неправильно расположены
    + Столбцы отображают не правильную инфрмацию
    + Столбцы неправильно группируют информацию
    + Столбцы форматируют значения неправильным цветом, шрифтом
2. Ссылка на Историю добавления или изменение функциональности, в которой наблюдается ошибка
3. `[Файл лога]`

### Запрос на исправление ошибки в Отчете
1. В ПМ "<Название ПМ>", в отчете "<Название отчета>", наблюдается ошибочное поведение Системы:

## Подзадача
### Область навигации
#### Добавить ссылку
1. В области навигации **добавить** ссылку <Полный/путь/к/ссылке> на <`[Название ПМ/]`Название представления>
2. Ссылка на Историю запрашиваемой функциональности

#### Изменить ссылку
1. В области навигации **изменить** ссылку <Полный/путь/к/ссылке> на <`[Название ПМ/]`Название представления>
2. Ссылка на Историю запрашиваемой функциональности

#### Удалить ссылку
1. В области навигации **удалить** ссылку <Полный/путь/к/ссылке>
2. Ссылка на Историю запрашиваемой функциональности

### Представление
#### Панель действий
##### Добавить действие
1. В представлении "<Название представления>", в панели действий, **добавить** действие «Полный/Путь/Название действия»
2. Ожидаемое поведение Системы при нажатии Пользователя на действие
3. Требования к внешнему виду:
    + Иконка
    + Ширина
    + Выравнивание
4. Ссылка на Историю запрашиваемой функциональности

##### Изменить действие
1. В представлении "<Название представления>", в панели действий, **изменить** действие «Полный/Путь/Название действия»
2. Описание ожидаемого поведения Системы при нажатии Пользователя на действие, в котором **выделены изменения**
3. Требования к внешнему виду:
    + Иконка
    + Ширина
    + Выравнивание
4. Ссылка на Историю запрашиваемой функциональности

##### Удалить действие
1. В представлении "<Название представления>", в панели действий, **удалить** действие «Полный/Путь/Название действия»
2. Ссылка на Историю запрашиваемой функциональности

#### Параметры отбора документов
1. В представлении "<Название представления>" реализовать отбор документов по условию:
  + Тип документа = <Тип_документа>
  + Тип_документа.<Название атрибута> = <Значение>
2. Ссылка на Историю запрашиваемой функциональности

#### Параметры отображения столбца
1. В представлении "<Название представления>" (добавить|изменить|удалить) столбец <Название столбца>:
  + Позиция: <Указать номер>
  + Значение: Тип_документа.<Название атрибута>
  + Форматирование:
    - Ширина
    - Размер шрифта
    - Цвет
2. Ссылка на Историю запрашиваемой функциональности
